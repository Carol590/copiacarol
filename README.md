# Predictor de Primas y Siniestros - Seguros Colombia

Aplicación web para predecir primas y siniestros mensuales usando XGBoost.

## Características
- Predicciones por compañía, ciudad y ramo
- Visualizaciones interactivas con Plotly
- Cálculo de ratios siniestrales
- Análisis de participación de mercado
